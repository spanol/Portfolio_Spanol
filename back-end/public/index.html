Documentação do back-end

Rota home

  • Rota: "/" retorna a documentação do back-end

Projects route

  • Rota: "/home/projects" retorna um painel de crud de projetos
  • Rota: (post)"/projects" é o create do crud, cria um projeto do banco de dados
  • Rota: (get)"/projects" é o read do crud, retorna um json com todos projetos salvos no banco de dados
  • Rota: (put)"/projects" é o update do crud,atualiza o projeto no banco de dados
  • Rota: (delete)"/projects" é o delete do crud, deleta um projeto do banco de dados

  • Rota: (get)"/contacts" é o read do crud, retorna um json com todos contatos salvos no banco de dados
  • Rota: (put)"/contacts" é o update do crud, atualiza o contato no banco de dados
  • Rota: (delete)"/contacts" é o delete do crud, deleta um projeto do banco de dados
